Bookmarklet

Add the bookmarklet below to your mobile device's web browser. When you visit a page that you want to pop open in a new Chrome tab, just push the bookmarklet!

Push Pop

Options

The identification token is used by the browser extension and the bookmarklet to keep their pushing and their popping on the same page.

Be sure to update your bookmarklet if you change the token.

The server is used as a middle man between the browser extension and the bookmarklet. Unless you've set up your own server, you can probably just leave this setting alone.
